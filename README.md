# random_nanoka
## The worst random number generator in the world, nanodesu.

## How does it work?
### There's a vec of anime characters who say nanoka a lot.
```rust
        let nanoka_characters = vec![
            "Hachikuji Mayoi",
            "Araragi Koyomi",
            "Senjougahara Hitagi",
            "Kanbaru Suruga",
            "Hanekawa Tsubasa",
            "Oshino Shinobu",
        ];
```
### And it gets the length of their names.
```rust
nanoka_characters.len()
```
